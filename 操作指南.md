## 📝 修改内容示例

### 1️⃣ 修改价格信息
找到类似这样的内容：
```javascript
"观山海": {
    huxing: `95平3房 700万<br>110平4房 820万`
}
```
修改为：
```javascript
"观山海": {
    huxing: `95平3房 750万<br>110平4房 870万`
}
```

### 2️⃣ 修改配套信息
```javascript
"观山海": {
    peitao: `项目配套内容...`
}
```
直接修改文字内容即可。

### 3️⃣ 修改价格区间页面
找到 `_PRICE_MAP` 部分：
```javascript
"_PRICE_MAP": `
    <tr><td><strong>观山海 (前海)</strong></td>
    <td>95平3房 700万<br>110平4房 820万</td>
    <td>低容积率2.66</td></tr>
`
```
修改价格数字即可。

## 🔧 常见问题

### Q1: 修改后手机看不到更新？
**解决方案：**
1. 等待 2-3 分钟（GitHub Pages 需要时间部署）
2. 强制刷新手机浏览器（下拉刷新或关闭重开）
3. 清除浏览器缓存

### Q2: 批处理文件运行失败？
**解决方案：**
1. 确保电脑已安装 Git
2. 确保网络连接正常
3. 检查是否已配置代理（如果需要）

### Q3: 提交时提示错误？
**解决方案：**
1. 检查文件是否正确保存
2. 检查是否删除了重要的符号
3. 使用备份恢复（见下方备份方法）
