# 价格自动更新系统 - 使用指南

## 🎯 系统概述

这套价格自动更新系统包含三个工具：

1. **价格管理.html** - 可视化管理价格
2. **价格更新助手.bat** - 快速更新助手
3. **一键提交并推送.bat** - 自动提交到GitHub

---

## 📝 方式一：使用价格管理工具（推荐）

### 步骤：

#### 1️⃣ 打开价格管理工具
```
双击：价格管理.html
```

#### 2️⃣ 更新价格
- 选择区域（如：南山区）
- 选择楼盘（如：观山海）
- 输入新的价格信息：
  ```
  95平3房 750万
  110平4房 870万
  ```
- 点击"更新价格"按钮

#### 3️⃣ 提交到GitHub
- 点击"一键提交到GitHub"按钮
- 或运行"一键提交并推送.bat"

#### 4️⃣ 等待部署
- 等 1-3 分钟
- 手机刷新查看

## 🔧 方式二：手动更新（高级）

### 直接修改JS文件

#### 1️⃣ 找到对应文件
```
南山区 → js\data_core.js
宝安区 → js\data_baoshan.js
福田区 → js\data_futian.js
龙华区 → js\data_longhua.js
光明区 → js\data_guangming.js
沙井 → js\data_shajing.js
```

#### 2️⃣ 找到价格区间部分
搜索：`_PRICE_MAP`

#### 3️⃣ 修改价格
找到类似这样的内容：
```html
<tr><td><strong>观山海 (前海)</strong></td>
<td>95平3房 700万<br>110平4房 820万</td>
<td>低容积率2.66</td></tr>
```

修改为：
```html
<tr><td><strong>观山海 (前海)</strong></td>
<td>95平3房 750万<br>110平4房 870万</td>
<td>低容积率2.66</td></tr>
```

#### 4️⃣ 保存并提交
- 保存文件（Ctrl+S）
- 双击"一键提交并推送.bat"

## ⏰ 定时任务设置（可选）

### Windows 任务计划程序

如果你希望每天自动检查更新，可以设置Windows任务计划：

#### 创建定时任务：

**方式A：使用价格更新助手（每天提醒）**

1. 打开"任务计划程序"
   - Win键 → 搜索"任务计划程序"

2. 创建基本任务
   - 点击"创建基本任务"
   - 名称：`价格更新提醒`
   - 触发器：`每天 23:00`
   - 操作：`启动程序`
   - 程序：`C:\Users\11465\Desktop\sz-realestate\价格更新助手.bat`

3. 完成
   - 点击"完成"

**效果：**
- 每天23点自动弹窗提醒你更新价格
- 打开价格管理工具
- 显示待更新的楼盘

## 💡 常见使用场景

### 场景1：单个楼盘价格调整
```
1. 打开"价格管理.html"
2. 选择区域和楼盘
3. 修改价格
4. 点击"更新价格"
5. 点击"一键提交到GitHub"
```

### 场景2：批量更新多个楼盘
```
1. 准备JSON格式数据：
   {
     "南山区": {
       "观山海": {
         "价格": ["95㎡：750万", "110㎡：870万"]
       },
       "绿景白石洲": {
         "价格": ["110㎡：1050万", "187㎡：2600万"]
       }
     }
   }

2. 打开"价格管理.html"
3. 选择"批量操作"卡片
4. 粘贴JSON数据
5. 点击"批量更新"
6. 点击"一键提交到GitHub"
```

### 场景3：每天例行检查
```
1. 设置Windows定时任务（每天23点）
2. 自动打开价格更新助手
3. 查看当天需要更新的楼盘
4. 使用价格管理工具更新
5. 一键提交到GitHub
```

## ⚠️ 注意事项

### 更新价格时：

1. **格式统一**
   ```
   正确：95㎡：750万
   正确：95-105㎡：750-870万
   错误：95万750（没有面积）
   ```

2. **换行使用 `<br>`**
   ```html
   95㎡：750万<br>110㎡：870万
   ```

3. **保持HTML结构**
   ```html
   <tr>
     <td><strong>楼盘名</strong></td>
     <td>价格信息</td>
     <td>卖点</td>
   </tr>
   ```

### 提交前检查：

- ✅ 文件已保存
- ✅ 价格格式正确
- ✅ HTML结构完整
- ✅ 浏览器查看正常

## 🆘 遇到问题？

### Q1: 提交后手机看不到更新？
**A:**
- 等待 2-3 分钟
- 强制刷新手机浏览器
- 清除浏览器缓存

### Q2: 价格格式不对？
**A:**
- 查看其他楼盘的格式
- 保持一致的风格
- 使用 `<br>` 换行

### Q3: 删除了重要的内容？
**A:**
- 查看 GitHub 历史版本
- 使用"一键备份.bat"恢复
- 或从上一个版本恢复
