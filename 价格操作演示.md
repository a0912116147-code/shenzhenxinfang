## ⚠️ 常见错误及解决

### 错误1：价格格式不对
```
❌ 错误：95750（没有单位）
✅ 正确：95㎡：750万

解决：
• 必须包含面积和价格
• 使用":"或"："分隔
• 价格必须带"万"字
```

### 错误2：提交后看不到更新
```
原因：
• Git没有成功推送
• 浏览器缓存

解决：
• 等待2-3分钟
• 强制刷新手机浏览器（关闭重开）
• 清除浏览器缓存
```

### 错误3：JSON格式错误
```
❌ 错误：
{
  "南山区": "观山海": {...}
}

✅ 正确：
{
  "南山区": {
    "观山海": {...}
  }
}

解决：
• 注意花括号配对
• 注意逗号位置
• 使用JSON验证工具
```
